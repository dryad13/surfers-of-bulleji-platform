# Content Strategy

## Positioning

Primary positioning:

> Surfers of Bulleji is a grassroots surf community from Karachi's coast, building Pakistan's surf future through self-taught skill, shared boards, and love for the sea.

Do not position the community as a charity case.

The story should feel:

```txt
Raw
Human
Athletic
Hopeful
Cinematic
Karachi-rooted
Community-first
Dignified
```

## Words to use

```txt
self-taught surfers
grassroots surf community
coastal community
Pakistan's surf future
from Bulleji to the world
shared boards
repaired boards
surf movement
community-built
sea-born
```

## Words to avoid

```txt
poor boys
helpless kids
slum surfers
impoverished children
save them
charity case
victims
```

It is acceptable to mention resource constraints, but not as the identity of the community.

## Core homepage message

```txt
Pakistan's first surf community, born on the shores of Karachi.
```

Supporting copy:

```txt
Self-taught surfers from Bulleji are building Pakistan's surf future with repaired boards, shared gear, and relentless love for the sea.
```

## Story arc

The platform should follow this emotional arc:

```txt
Place -> People -> Struggle -> Skill -> Dream -> Support -> Impact
```

### Place

Bulleji / Hawksbay, Karachi. The sea is not a backdrop; it is part of their identity.

### People

Young surfers, fishermen, students, lifeguards, community members, mentors.

### Struggle

Limited equipment, broken boards, lack of infrastructure, little institutional support.

### Skill

Self-taught, persistent, physically brave, sea-aware.

### Dream

Recognition, safer training, better equipment, competitions, surf tourism, and a future pathway for Pakistan surfing.

### Support

Make help practical and specific.

### Impact

Show what changed: boards repaired, sessions held, gear received, youth trained, stories published.

## Page copy direction

### Home

Goal: immediate emotional clarity.

Tone: cinematic, short, confident.

Avoid long paragraphs above the fold.

### Story

Goal: explain origin and dignity.

Tone: documentary style.

### Support

Goal: convert emotion into specific action.

Tone: clear and practical.

Use support cards:

```txt
Sponsor a Board
Repair a Board
Donate Safety Gear
Donate Surf Gear
Sponsor Training
Partner With the Community
```

### Boardroom

Goal: make support visible.

Tone: tangible and transparent.

Use language like:

```txt
Every board has a story. Some are teaching beginners. Some need repair. Some are still missing.
```

### Book / Request a Session

Goal: create earned-income path.

Tone: exciting but safe.

Use:

```txt
Request a Surf Session
```

Do not use:

```txt
Guaranteed booking
Instant confirmation
Pay now
```

### Media

Goal: legitimacy and press readiness.

Tone: credible, concise.

## Prototype disclaimer

Use this before approval:

```txt
Concept MVP prepared for review by Surfers of Bulleji. Final launch, donation links, profiles, images, and booking flows require community approval.
```

## Safety disclaimer

Use on session request page:

```txt
All surf sessions depend on sea conditions, weather, instructor availability, and participant swimming ability. Submission of this form does not confirm a session. The team will review and confirm manually on WhatsApp.
```

## Donation/support disclaimer

Use before official approval:

```txt
This prototype does not collect payments. Support inquiries are for demonstration and review only until official channels are approved by the community.
```

## Press/media language

Use:

```txt
For journalists, filmmakers, photographers, and partners interested in documenting or supporting Pakistan's emerging surf movement.
```

## Approval workflow

Before launch, every content item should be marked as:

```txt
placeholder
public-source
community-approved
```

Only `community-approved` content should be treated as official.
